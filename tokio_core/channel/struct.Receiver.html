pub struct Receiver<T> { /* fields omitted */ }

The receiving half of a channel used for processing messages sent by a Sender.

A Receiver cannot be cloned, so only one thread can receive messages at a time.

This type is created by the channel function and implements the Stream trait to represent received messages.
